Business-rule notes

  • Sales = SUM(COALESCE(order_total, 0)). Orders = COUNT(DISTINCT order_item_info).
  • Sales Change / Order Change = (Current − Previous) ÷ Previous, shown as a percentage. Undefined ("—") when Previous = 0.
  • Trend is Sales-based only: UP / DOWN / NO CHANGE. Previous = 0 and Current > 0 → UP. Previous = 0 and Current = 0 → NO CHANGE.
  • Achieve Sales % / Achieve Order % measure progress against 130% of the equivalent prior-year period. Shown as "—" (undefined) when the prior-year base is zero — no percentage is fabricated for that case.
  • "NOT IMPROVED" appears only when both Previous and Current are zero for a row; it is never used as a Trend value.
  • Total rows are computed from aggregate totals, never from averaging row-level percentages.
